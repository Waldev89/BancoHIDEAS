Banco de Hideas

Es un Sistema creado para incrementar los niveles de innovación dentro de la compañía, muchos se preguntarán por que el nombre de Hideas con “H”, se denominó así porque es un Sistema de ideas para Heuristica. Allí se van a poder almacenar las ideas de los empleados, las cuales solo van a ser visibles para el usuario que crea la idea y el comité, la idea solo se puede borrar mientras está se encuentre en estado borrador, una vez se le hagan los cambios iniciales correctamente, se procede a consignar la idea para que el comité pueda actuar sobre ella.
